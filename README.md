# sonata
A serverless Discord bot based on CloudFlare Workers for BotiCord.top

## How work?
* Requires only `applications.commands` scope
* Not use node packages (only for development)
* Zero-time ping
* Absolutely free!

## But?
* Rate limit for Cloudflare IPs
